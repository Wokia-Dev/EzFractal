# EzFractal

## install
```py
pip install -r requirements.txt
```

## run 
```py
python main.py
```
## References
[EZ](https://github.com/Wokia-Dev/EZ)

## Basic Control
- r : reset 
- crtl + s : export
- ←↑→↓ : move in fractal
- scroll : zoom in/out
- echap : quit
